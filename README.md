Routes included
1. index.ejs:
contains product categories
2. about.ejs (/about):
Contains other store locations with address and managers
3. register.ejs (/register and /registered):
allows users to register with first name, last name and email
4. search.ejs (/search)
searches through product categories
5. survey.ejs (/survey)
provides a form to fill out with first name, last name, email, age and asks questions.
6. survey-response.ejs (survey-response)
submits form and displays users answers to the form
